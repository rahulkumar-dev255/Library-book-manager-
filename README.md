# 📚 Library Book Manager

A modern and user-friendly Library Book Management System built using JavaScript, HTML, and CSS. This application helps users manage library books efficiently by allowing them to add, view, search, and organize book records through an intuitive interface.

## 🚀 Live Demo

🔗 https://rahulkumar-dev255.github.io/Library-book-manager/

---

## 📖 Overview

Library Book Manager is a web-based application designed to simplify book management tasks. It provides an easy way to maintain book records and improves accessibility for users managing a small library or personal book collection.

The project demonstrates core JavaScript concepts such as DOM manipulation, event handling, data management, and responsive UI development.

---

## ✨ Features

- ➕ Add new books to the library
- 📋 View all available books
- 🔍 Search books quickly
- 🗑️ Remove books from the collection
- 📱 Responsive design for different screen sizes
- ⚡ Fast and lightweight user interface
- 🎯 User-friendly experience

## 🖥️ Installation

1. Clone the repository

```bash
git clone https://github.com/rahulkumar-dev255/Library-book-manager.git
```

2. Navigate to the project folder

```bash
cd Library-book-manager
```

3. Open `index.html` in your browser.
